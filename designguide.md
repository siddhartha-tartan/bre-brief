# UI Basics

Use these as the default choices when building new screens, unless the existing screen already follows a different pattern.

## 1. Typography

- Use `Manrope` for the app.
- For most UI text, use:
  - `16px` for main button text and important labels
  - `14px` for input labels, field text, helper text, and table content
  - `12px` for smaller table headers or metadata
- Common font weights:
  - `400` for normal body text
  - `500` for labels and buttons
  - `600` for headings and emphasis

## 2. Buttons

- Primary button:
  - Background: black `#000000`
  - Text: white `#FFFFFF`
  - Border radius: `10px`
  - Height: `45px`
- Secondary button:
  - Background: white `#FFFFFF`
  - Text: dark gray `#344054`
  - Border: `1px solid #D0D5DD`
  - Border radius: `10px`
  - Height: `45px`
- Branded call-to-action button:
  - Blue gradient from `#3762DD` to `#1D3577`
- Keep icon and text spacing around `10px`.

## 3. Inputs and Search

- Standard input height: `40px`
- Standard input border radius: `8px` to `10px`
- Default border: light gray
- Active/focus color: blue `#3762DD`
- Placeholder text: muted gray `#ABAAAD`
- Search bars usually use:
  - White background
  - `10px` radius
  - `16px 20px` padding

## 4. Colors

- Main accent blue: `#3762DD`
- Dark blue used in gradients: `#1D3577`
- Main text: `#101828`
- Secondary text: `#344054` or `#555557`
- Page/card backgrounds: white or very light gray like `#F9FAFB` / `#FEFEFF`
- Border color: `#D0D5DD` or `#E4E7EC`
- Success: green `#12B76A`
- Error: red `#F04438`

## 5. Cards and Containers

- Card background: white
- Card border: `1px solid #E4E7EC`
- Card border radius: `16px`
- Card padding: usually `24px`
- Use very light shadows only, not heavy shadows.

## 6. Tables

- Header background: very light gray `#FAFAFA`
- Header height: `56px`
- Header text: small, bold, muted dark gray
- Body rows usually alternate between white and very light gray
- Table header corners are softly rounded, around `10px`

## 7. Spacing

- `24px` is the most common section/card padding
- `16px` is common for smaller spacing blocks
- `10px` is the common gap between icons and text, or between closely related items

## 8. Special Case

- HR Portal screens use a maroon accent instead of blue:
  - Primary: `#7D152B`
  - Lighter maroon: `#B3475D`
